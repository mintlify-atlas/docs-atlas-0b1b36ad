## What is CURD?

CURD is a code-intelligence control plane for humans and agents. It helps you understand a codebase structurally, mutate code safely inside a shadow workspace, inspect graph impact before committing changes, and author repeatable workflows in `.curd` source scripts.

## Quick Start

<Steps>
  <Step title="Install CURD">
    Choose your preferred installation method:
    
    <CodeGroup>
    ```bash macOS / Linux (Homebrew)
    brew tap aerospeedsta/curd && brew install curd
    ```
    
    ```bash Windows (Winget)
    winget install curd
    ```
    
    ```bash Python (uvx)
    uvx --from curd-python curd --version
    ```
    
    ```bash Node.js (bunx)
    bunx --bun curd-node --version
    ```
    </CodeGroup>
  </Step>
  
  <Step title="Initialize a workspace">
    From your project root, initialize CURD:
    
    ```bash
    curd init
    curd doctor .
    ```
  </Step>
  
  <Step title="Start exploring">
    Use CURD's code intelligence tools:
    
    ```bash
    curd search alpha
    curd read src/lib.rs::alpha
    curd graph src/lib.rs::alpha --depth 2
    ```
  </Step>
</Steps>
